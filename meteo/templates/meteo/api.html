API Meteo

Esta API fornece informações meteorológicas. Abaixo estão os detalhes dos endpoints disponíveis:

Endpoints

1. Lista de Cidades

URL: /api/cidades/

Método: GET

Retorna a lista de cidades para as quais existem previsões meteorológicas.

Exemplo de resposta:
[
    {
        "idRegiao": 1,
        "idAreaAviso": "AVR",
        "idConcelho": 5,
        "globalIdLocal": 1010500,
        "latitude": "40.6413",
        "idDistrito": 1,
        "local": "Aveiro",
        "longitude": "-8.6535"
    },
    ...
]

2. Previsão Meteorológica de Hoje para uma Cidade

URL: /api/previsao-hoje/<global_id>/

Método: GET

Retorna a previsão meteorológica de hoje para a cidade especificada pelo global_id.

Parâmetros:
- global_id: ID global da cidade

Exemplo de resposta:
{
    "cidade": "Lisboa",
    "temp_min": 16.2,
    "temp_max": 23.4,
    "data": "2024-06-10",
    "descricao_tempo": "Céu parcialmente nublado",
    "precipitacao": 0,
    "icon_url": "https://api.ipma.pt/open-data/icons/3.svg"
}

3. Previsão Meteorológica para os Próximos 5 Dias para uma Cidade

URL: /api/previsao-5-dias/<global_id>/

Método: GET

Retorna a previsão meteorológica para os próximos 5 dias para a cidade especificada pelo global_id.

Parâmetros:
- global_id: ID global da cidade

Exemplo de resposta:
[
    {

        "precipitacao": 0,

    ...
